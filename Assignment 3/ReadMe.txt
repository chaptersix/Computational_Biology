Describe how to run your program.


Project was developed on a Windows Professional system running python 3.7.5

to run the project:
1. cd into the parent directory (the directory containing this file)
2. run the following command
    python classify.py <path_to_fasta_file>

    Example: python classify.py fasta/1a3a.fasta


classify.py requires the following:
    the pssm folder containing the all of the .pssm files
    means_variances.txt
    total_probs.txt
